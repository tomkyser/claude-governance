### Critical Bug Found: F17 — parentMessage.message.id
Write, edit, and read ALL crashed with `TypeError: undefined is not an object (evaluating '$?.message.id')`. Root cause: CC's FilePersistence feature accesses `parentMessage.message.id` via optional chaining. Our initial mock `{ uuid }` was WORSE than passing nothing — when `$` is non-null but missing `.message`, the chain doesn't short-circuit. Fix: `{ uuid, message: { id, role, content } }`.

### G10 Regression Found
`for await (const x of []) {}` gives V8 error "Unexpected reserved word" — no "await" in message. Error-message matching missed it. Fixed by switching to script-source checking (`/\bawait\b/.test(script)`).

### Embedded Search Gap Identified: G15
REPL's grep/glob use system `/usr/bin/grep` and `/usr/bin/find`, NOT the embedded ugrep/bfs/rg compiled into the CC binary. Tracked as G15 in 2b-gaps-2.
